# Audio samples

The Piano instrument uses sampled audio from the [Tonejs/audio](https://github.com/Tonejs/audio) repository, which hosts files used in official Tone.js examples.

## Collections used

- **Piano:** [casio](https://github.com/Tonejs/audio/tree/master/casio)  
  Sparse note set; Tone.Sampler repitches to cover the full riff range.  
  Base URL: `https://tonejs.github.io/audio/casio/`
